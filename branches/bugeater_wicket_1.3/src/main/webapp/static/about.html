About Bugeater

Intruduction

Bugeater is an open source software development issue tracker. This system can help organizations track bugs (defects) in software as well as requests for new functionallity and requests for changes in existing functionallity.

Bugeater was designed and built by Philip A. Chapman. The logo was designed by Michael Worcester. Creative input was provided by Andrew Lombardi and Igor Vaynberg.

The bugeater project is hosted at https://developer.berlios.de/projects/bugeater/ You can find the source code, mailing lists, and other forms of support there.
